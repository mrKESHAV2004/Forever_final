#FOREVER 

REQUIREMENTS FOR THIS PROJECT
1. Code Editor
2. Libraries mentioned in text file
3. Cloudinary Account
4. MongoDB

HOW TO RUN THIS PROJECT
1. Go to the .env file in the backend and file out neccesaary keys
2. Now go the terminal and open a dedicated terminal for each of the folders present
3. In the backend folder use {nodemon server.js} command to run the backend required
4. In case your port 4000 is in use make required changes in the .env files of frontend and backend
5. Now in the terminal for frontend folder run {npm run dev}
6. Now in the terminal for admin folder run {npm run dev}
